# Research-synthesis prompt

Use only fictional, public, or explicitly authorized material. Do not paste confidential, personal, client, employer, privileged, or restricted information into an external model. AI output is a draft classification, not source evidence or independent review.

```text
Using only the supplied, permission-cleared notes, build an evidence matrix.

For each claim:
- quote or closely preserve the supplied claim;
- classify it as fact, judgment, assumption, recommendation, or gap;
- cite the supplied source reference;
- name the weakest link;
- state what evidence would change the conclusion;
- mark any unsupported statement as an assumption.

After the matrix:
1. Name the most fragile claim.
2. List three fixable evidence gaps.
3. Draft a conditional decision rule.
4. List conflicts or missing context that require human review.

Do not invent sources, facts, source-check dates, checker identities, outcomes, or legal/compliance conclusions. If processing authorization is unclear, stop with: HUMAN REVIEW REQUIRED — DO NOT PASTE.

Notes:
<fictional, public, or authorized text only>
```

A responsible human compares the matrix against the original sources before using it in a decision packet.
