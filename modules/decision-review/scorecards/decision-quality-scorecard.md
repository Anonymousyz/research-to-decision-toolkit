# Decision-quality scorecard

Score each item 0 (absent), 1 (partial), or 2 (clear and evidenced).

| Item | Score |
|---|---:|
| Decision body and owner are named | |
| Default outcome and deadline are explicit | |
| At least three materially different alternatives are compared | |
| Reversibility and commitment boundary are explicit | |
| Fragile assumptions have tests, owners, and deadlines | |
| Pre-mortem covers at least five distinct failure causes | |
| Stop condition is measurable and invokable by a named role | |
| An independent human reviewer has responded to the strongest challenge | |

**Vetoes:** no named decision body; no default outcome; no stop condition for a high-consequence commitment; AI output presented as independent review or source evidence.

A high score cannot offset a veto.
